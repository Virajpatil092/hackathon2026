## API Integration

All these MCC codes and mappings are available through the following API endpoints:

- `GET /api/v1/carbon-footprint` - Calculate total footprint
- `GET /api/v1/carbon-footprint/categories` - Breakdown by category
- `GET /api/v1/carbon-footprint/mcc/distinct` - List all distinct MCCs
- `GET /api/v1/carbon-footprint/mcc/categories` - Group MCCs by category
- `GET /api/v1/carbon-footprint/mcc/stats` - Statistics about MCCs
